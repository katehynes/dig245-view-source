How to View Source Code in Google

  1. First, open Google and navigate to any website you want.
  2. Next, select "View" in the top menu bar of your computer. Choose "Developer" from the drop-down menu; it should be the last option.
  3. Once you select "Developer," select "View Source." This should open up the source code of the website in a new tab.




Try It Out

You can also access other helpful web development tools, or Dev tools, by right-clicking anywhere on the web page and selecting "Inspect." Mac users can also do this by using the "Command + Option + j" shortcut.





Other Web Dev Tools

  • Right-click on an element and select "Edit as HTML" to edit it live and see changes reflected on your website.
  • Right-click on an element and select "Delete element" to delete an element inside your source code.
  • Look for secret messages in the "Console" window after inspecting source code.
